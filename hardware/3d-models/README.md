# 3D Print Files

This folder contains STL files for the Satoshi Pet enclosure.

## Files

| File | Description |
|------|-------------|
| `case-top.stl` | Top shell with display cutout |
| `case-bottom.stl` | Bottom shell with battery compartment |
| `button-cap.stl` | Optional button extension |

## Print Settings

| Setting | Recommended |
|---------|-------------|
| Material | PLA or PETG |
| Layer Height | 0.2mm |
| Infill | 15-20% |
| Supports | Yes (for button cutout) |
| Walls | 2-3 perimeters |

## Assembly

1. Print all parts
2. Insert Heltec board into bottom shell
3. Route battery and buzzer wires
4. Snap or screw top shell in place
5. (Optional) Add button cap

## Customization

STEP files are provided for modification. You can:
- Adjust dimensions for different batteries
- Add ventilation holes
- Customize button placement
- Add belt clip or lanyard hole
